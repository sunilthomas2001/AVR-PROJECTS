# AVR Embedded Systems Projects

## Overview

This repository contains a collection of Embedded C projects developed using the **ATmega328P microcontroller**. The projects were implemented using direct register-level programming to gain practical experience with microcontroller peripherals, communication protocols, and embedded system development.

The repository covers fundamental embedded concepts such as GPIO, Timers, Interrupts, ADC, PWM, UART, and I2C communication through hands-on hardware-based projects.

---

## Tools & Technologies

* Microcontroller: ATmega328P
* Programming Language: Embedded C
* Development Environment: Microchip Studio
* Version Control: Git & GitHub
* Hardware Platform: Arduino Uno (ATmega328P)
